📌 Ejercicio 1: Mensaje de bienvenida personalizado

Función que muestra un mensaje de bienvenida personalizado.

def bienvenido(nombre):
    print("Bienvenido", nombre)

bienvenido("William")

📌 Ejercicio 2: Número par o impar

Función que determina si un número es par o impar.

def es_par_o_impar(numero):
    if numero % 2 == 0:
        return "Par"
    else:
        return "Impar"

print(es_par_o_impar(7))

📌 Ejercicio 3: Suma de dos números

Función que suma dos números.

def suma(num1, num2):
    return num1 + num2

print(suma(3, 6))

📌 Ejercicio 4: Longitud de una palabra

Función que retorna la longitud de una palabra.

def longitud_palabra(palabra):
    return len(palabra)

print(longitud_palabra("William"))

📌 Ejercicio 5: Cuadrado de un número

Función que calcula el cuadrado de un número.

def cuadrado(numero):
    return numero ** 2

print(cuadrado(7))

📌 Ejercicio 6: Mayor de tres números

Función que encuentra el mayor de tres números.

def mayor_tres_numeros(num1, num2, num3):
    return max(num1, num2, num3)

print(mayor_tres_numeros(2, 6, 9))

📌 Ejercicio 7: Celsius a Fahrenheit

Función que convierte grados Celsius a Fahrenheit.

def celsius_a_fahrenheit(celsius):
    return (celsius * 9/5) + 32

print(celsius_a_fahrenheit(30))

📌 Ejercicio 8: Suma de una lista

Función que suma todos los elementos de una lista.

def suma_lista(lista):
    return sum(lista)

print(suma_lista([2, 4, 6, 7, 8, 8]))

📌 Ejercicio 9: Número positivo o negativo

Función que determina si un número es positivo o negativo.

def es_positivo_o_negativo(numero):
    if numero > 0:
        return "Positivo"
    elif numero < 0:
        return "Negativo"
    else:
        return "Cero"

print(es_positivo_o_negativo(5))

📌 Ejercicio 10: Invertir una palabra

Función que invierte una palabra.

def invertir_palabra(palabra):
    return palabra[::-1]

print(invertir_palabra("Hola william"))
