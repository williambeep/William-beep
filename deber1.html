#1 COMPARACIÓN

📌 Pide un número al usuario y muestra si es mayor que 50

numero = int(input("Ingrese un número: ")) if numero > 50: print("El número es mayor que 50.") else: print("El número no es mayor que 50.")
#2 EDAD

📌 Pide la edad y muestra si es mayor de edad

edad = int(input("Ingrese su edad: ")) nombre = input("Ingrese su nombre: ") if edad >= 18: print(f"{nombre}, eres mayor de edad") else: print(f"{nombre}, no eres mayor de edad")
#3 MÚLTIPLO

📌 Pide un número y verifica si es múltiplo de 3

numero = int(input("Ingresa un número: ")) if numero % 3 == 0: print("Es múltiplo de 3") else: print("No es múltiplo de 3")
#4 CONTRASEÑA

📌 Prueba una contraseña y comprueba si es correcta

clave = input("Ingresa la contraseña: ") if clave == "William12345": print("Acceso correcto") else: print("Acceso denegado")
#5 NOTA

📌 Pide una nota y muestra si está aprobado o reprobado

nota = float(input("Ingrese su nota: ")) if nota >= 7: print("Está aprobado") else: print("Está reprobado")
#6 COMPARACIÓN

📌 Pide dos números y muestra cuál es mayor

num1 = float(input("Ingresa un número: ")) num2 = float(input("Ingresa otro número: ")) if num1 > num2: print("El primer número es mayor") elif num2 > num1: print("El segundo número es mayor") else: print("Los números son iguales")
#7 SIGNO

📌 Determina si un número es positivo, negativo o cero

numero = float(input("Ingrese un número: ")) if numero > 0: print("El número es positivo") elif numero < 0: print("El número es negativo") else: print("El número es cero")
#8 VOCAL

📌 Pide una letra e indica si es vocal

letra = input("Ingrese una letra: ") if letra in "aeiouAEIOU": print("Es una vocal") else: print("No es vocal")
#9 CLIMA

📌 Pide una temperatura y muestra si hace frío (menos de 12°C)

temperatura = float(input("Ingrese la temperatura: ")) if temperatura < 12: print("Está haciendo frío") else: print("No está haciendo frío")
#10 BISIESTO

📌 Pide un año e indica si es bisiesto

año = int(input("Ingresa un año: ")) if año % 4 == 0: print("Es un año bisiesto") else: print("No es bisiesto")

🤔 ¿Qué es la sentencia IF?

La sentencia IF (si en español) es una estructura de control fundamental en programación que permite tomar decisiones basadas en condiciones. Evalúa una expresión y ejecuta un bloque de código solo si la condición se cumple.

⚡ En Ciberseguridad, IF es esencial para:

  • 🔐 Validar credenciales de acceso
  • 🛡️ Verificar permisos de usuario
  • 📊 Analizar patrones de tráfico sospechoso
  • 🔒 Controlar autenticación multifactor
