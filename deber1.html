⚡ IF/ELSE
#1 COMPARACIÓN

📌 Verificar si un número es mayor que 50

numero = int(input("Ingrese un número: "))
if numero > 50:
    print("✅ El número es mayor que 50")
else:
    print("❌ El número no es mayor que 50")
#2 EDAD

📌 Verificar si una persona es mayor de edad

nombre = input("Ingrese su nombre: ")
edad = int(input("Ingrese su edad: "))
if edad >= 18:
    print(f"{nombre}, eres mayor de edad")
else:
    print(f"{nombre}, eres menor de edad")
#3 MÚLTIPLO

📌 Verificar si un número es múltiplo de 3

numero = int(input("Ingrese un número: "))
if numero % 3 == 0:
    print("✅ Es múltiplo de 3")
else:
    print("❌ No es múltiplo de 3")
#4 SEGURIDAD

📌 Validar contraseña de acceso

clave = input("Ingrese la contraseña: ")
if clave == "William12345":
    print("🔓 Acceso concedido")
else:
    print("🔒 Acceso denegado")
#5 CALIFICACIÓN

📌 Determinar si un estudiante aprueba o reprueba

nota = float(input("Ingrese su nota (0-10): "))
if nota >= 7:
    print("🎉 ¡Aprobado!")
else:
    print("📚 Reprobado")
#6 COMPARACIÓN

📌 Encontrar el mayor de dos números

num1 = float(input("Primer número: "))
num2 = float(input("Segundo número: "))
if num1 > num2:
    print(f"{num1} es mayor")
elif num2 > num1:
    print(f"{num2} es mayor")
else:
    print("Son iguales")
#7 SIGNO

📌 Clasificar número en positivo, negativo o cero

numero = float(input("Ingrese un número: "))
if numero > 0:
    print("✅ Positivo")
elif numero < 0:
    print("📉 Negativo")
else:
    print("⚪ Cero")
#8 VOCAL

📌 Verificar si una letra es vocal o consonante

letra = input("Ingrese una letra: ").lower()
if letra in "aeiou":
    print("🔊 Vocal")
else:
    print("📝 Consonante")
#9 CLIMA

📌 Determinar clima según temperatura

temp = float(input("Temperatura (°C): "))
if temp < 12:
    print("❄️ Frío")
elif temp > 25:
    print("☀️ Calor")
else:
    print("🌡️ Agradable")
#10 CALENDARIO

año = int(input("Ingrese un año: "))

    print("📅 Año bisiesto")
else:
    print("📅 No es bisiesto")

🤔 ¿Qué es la sentencia IF?

La sentencia IF (del inglés "si") es una estructura de control fundamental que permite tomar decisiones en un programa. Evalúa una condición y ejecuta un bloque de código si esta se cumple.

⚡ En Ciberseguridad, la sentencia IF es esencial para:

  • Validar credenciales de acceso
  • Verificar permisos de usuario
  • Analizar patrones de tráfico sospechoso
  • Controlar autenticación multifactor
  • Detectar intentos de intrusión
