📌 Ejercicio 1: Números del 1 al 10

Imprime los números del 1 al 10 usando un bucle for.

for i in range(1, 11):
    print(i)

📌 Ejercicio 2: Números pares del 1 al 20

Imprime los números pares del 1 al 20.

for i in range(1, 21):
    if i % 2 == 0:
        print(i)

📌 Ejercicio 3: Tabla de multiplicar

Muestra la tabla de multiplicar del número ingresado.

numero = int(input("Ingrese un número: "))
for i in range(1, 13):
    print(f"{numero} x {i} = {numero * i}")

📌 Ejercicio 4: Números descendentes

Imprime los números del 10 al 1 en orden descendente.

for i in range(10, 0, -1):
    print(i)

📌 Ejercicio 5: Suma acumulada

Suma acumulada desde el 1 hasta un número dado.

numero = int(input("Ingrese un número: "))
suma = 0
for i in range(1, numero + 1):
    suma += i
print(f"La suma es: {suma}")

📌 Ejercicio 6: Contar vocales

Cuenta cuántas vocales hay en una palabra.

palabra = input("Ingrese una palabra: ")
contador = 0
for letra in palabra:
    if letra in "aeiouAEIOU":
        contador += 1
print(f"La cantidad de vocales es: {contador}")

📌 Ejercicio 7: Validar contraseña

Solicita una contraseña hasta que sea correcta.

clave_correcta = "William12345"
clave = ""

while clave != clave_correcta:
    clave = input("Ingrese la contraseña: ")
    if clave != clave_correcta:
        print("Incorrecta, intente de nuevo")

print("Acceso autorizado.")

📌 Ejercicio 8: Múltiplos de 5

for i in range(1, 51):

        print(i)

📌 Ejercicio 9: Número mayor

Pide 5 números e indica el mayor.

mayor = None

for i in range(5):
    numero = float(input(f"Ingresa el número {i+1}: "))
    if mayor is None or numero > mayor:
        mayor = numero

print(f"El número mayor es: {mayor}")

📌 Ejercicio 10: Triángulo de asteriscos
